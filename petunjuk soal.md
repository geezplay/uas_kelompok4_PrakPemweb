Assalamu'alaikum, berikut adalah perintah untuk 
TUGAS UJIAN AKHIR SEMESTER (UAS)
Mata Kuliah: Pemrograman Web
Ketentuan Tugas

Setiap mahasiswa wajib membuat 1 (satu) project aplikasi secara individu dengan ketentuan sebagai berikut:

Project yang dibuat harus berbeda dengan project mahasiswa lainnya.
Project minimal memiliki fitur CRUD (Create, Read, Update, Delete).
Bebas menggunakan bahasa pemrograman, framework, database, maupun teknologi pendukung lainnya.
Project harus dapat di-deploy sehingga dapat diakses melalui internet.
Source code project wajib diunggah ke repository GitHub.
Video Presentasi

Setiap mahasiswa wajib membuat video presentasi yang memuat:

Menampilkan wajah (face camera) selama presentasi.
Menjelaskan PowerPoint (PPT) yang berisi:
Latar belakang project.
Tujuan pembuatan aplikasi.
Tools, framework, bahasa pemrograman, database, dan teknologi yang digunakan.
Arsitektur atau alur kerja aplikasi.
Fitur-fitur yang tersedia.
Mendemonstrasikan aplikasi yang telah dibuat, termasuk seluruh proses CRUD.
Menunjukkan bahwa aplikasi telah berhasil di-deploy dan dapat diakses melalui browser (gunakan vercel).
Menampilkan repository GitHub yang berisi source code project.
Berkas yang Dikumpulkan

Unggah melalui Google Classroom dengan format berikut:

Link website hasil deployment.
Link repository GitHub.
Link video presentasi (Google Drive/YouTube Unlisted atau media lain yang dapat diakses).
File PowerPoint (PPT) dalam format .pptx atau .pdf.
Ketentuan Penilaian

Penilaian akan mempertimbangkan aspek-aspek berikut:

Kelengkapan fitur CRUD.
Orisinalitas project.
Kesesuaian penggunaan teknologi.
Keberhasilan deployment aplikasi.
Kerapihan dan kualitas source code.
Kemampuan menjelaskan project pada saat presentasi.
Kelengkapan dokumen dan berkas yang dikumpulkan.
Catatan:
Tugas dikerjakan secara *Berkelompok*, sesuai dengan kelompok yang sudah dilist.
Dilarang mengumpulkan project yang sama atau melakukan plagiarisme. Kesamaan project maupun source code akan menjadi pertimbangan dalam penilaian.
Pastikan seluruh link yang dikumpulkan dapat diakses sebelum batas waktu pengumpulan.
Pengumpulan dilakukan melalui Google Classroom sesuai dengan jadwal yang telah ditentukan. Untuk deadline pengumpulan nya tanggal 8 Juli 2026 jam 23.59. Jika terlambat maka di anggap tidak mengerjakan. Yang mengumpulkan cukup ketua kelompok nya saja dengan menulis nama anggota dan NPM  di komentar.
